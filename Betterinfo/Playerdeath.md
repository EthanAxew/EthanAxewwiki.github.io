# 玩家死亡消息

------

### 一.关于配置文件

1.玩家可以自定义是否开启死亡消息更改！

```yml
deathmessages:
  #自定义消息开关
  enable: true
  #多行死亡提示
  list:
    enable: true
```

开启多行显示,才可以多行哦！

------

### 二.关于语言文件

```yml
deathmessages: "[§d💀§f] §8玩家§7%player_name%§8与世长辞！"
deathmessageslist:
  - "§8让我们为他哀悼!"
  - "xxx"
  - "xxx"
  - "xxx"
```

如果需要多行请按照格式填写,一定注意格式！
