# 玩家加入消息

------

### 一.关于配置文件

1.玩家可以自定义是否开启加入消息更改(如果和其他插件冲突请关闭,如CMI,ESS)

```yml
#加入消息开关
joinmessages:
  enable: true
  #多行显示
  list:
    enable: true
```

开启多行显示,才可以多行哦！

------

### 二.关于语言文件

```yml
#关于玩家加入/退出消息自定义
joinmessages: "[§b✔§f] §b玩家§f%player_name%§b加入了游戏"
joinmessageslist:
  - "§6让我们一起欢迎他"
  - "xxx"
  - "xxx"
```

如果需要多行请按照格式填写,一定注意格式！
