# 权限

------

- 本插件不需要配置任何权限,服主可以直接使用(热加载可能导致默认权限无效)！

### 权限：

| 权限             | 用途                                                         | 默认拥有       |
| ---------------- | ------------------------------------------------------------ | -------------- |
| betterinfo.admin | 使用reload,help权限<br />(由于插件命令就只有一个就设置了help为op拥有) | op             |
| betterinfo.info  | 使用/binfo<br />(用来获得查询物品)                           | op<br />player |
